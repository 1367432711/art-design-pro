---
name: git push 规则
description: 禁止自动推送代码，必须等用户明确说"推送"时才执行
type: feedback
---

**规则**: 不要在 git commit 后自动执行 git push

**为什么**: 用户希望手动控制推送时机，可能需要在推送前检查代码或做其他操作

**如何应用**:

- git commit 后**不要**自动执行 git push
- 只有当用户明确说 "推送" 或 "push" 时，才执行 git push
- 如果需要告知用户可以推送了，用文字提示即可，不要自动执行

**示例**:

- ✅ 用户说："推送" → 执行 git push
- ❌ commit 完成后自动推送 → 禁止
- ❌ 用户没说推送就主动执行 → 禁止
