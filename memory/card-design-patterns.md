# 卡片设计模式库

## 1. 用户资料卡片 (Profile Card)

### 结构布局

```
┌─────────────────────────────────┐
│  封面图区域 (height: 120px)      │
│  ┌─────┐                        │
│  │头像 │                        │
│  └─────┘                        │
├─────────────────────────────────┤
│  用户名 [认证] 🟢 在线状态       │
│  职位标题                        │
├─────────────────────────────────┤
│  ┌─────┬───────┬─────────┐      │
│  │统计 │ 统计  │ 统计    │      │
│  └─────┴───────┴─────────┘      │
├─────────────────────────────────┤
│  ┌─────────────────────────┐    │
│  │    CTA 按钮              │    │
│  └─────────────────────────┘    │
└─────────────────────────────────┘
```

### 设计规范

**容器规格**

- 宽度：380-420px
- 圆角：24-32px 大圆角
- 背景：纯白 #FFFFFF
- 阴影：柔和扩散阴影 `0 4px 24px rgba(0,0,0,0.08)`
- 内边距：24px

**顶部区域 - 封面 + 头像**

- 封面图高度：100-120px
- 封面背景：浅蓝渐变抽象图形
- 封面圆角：与卡片顶部圆角一致
- 头像尺寸：64-72px 圆形
- 头像位置：封面图下方，向左偏移 24px
- 头像边框：4px 白色描边

**用户信息区**

- 用户名：20-22px, 字重 700, 颜色 #1A1A1A
- 认证徽章：16px, 紫色渐变圆形 + 白色星形
- 在线状态：8px 绿色圆点 + "Online" 文字 (#6B7280)
- 职位标题：16px, 字重 400, 颜色 #9CA3AF
- 间距：用户名与职位 6-8px

**统计数据区**

- 容器：浅灰蓝背景 (#F8FAFC), 圆角 16px
- 内边距：20px 16px
- 三列均分布局，分隔线：1px #E2E8F0
- 数值：24px, 字重 600, 颜色 #1A1A1A
- 标签：14px, 字重 400, 颜色 #9CA3AF
- 居中对齐

**CTA 按钮**

- 宽度：100%
- 高度：48-56px
- 背景：纯黑 #1A1A1A
- 文字：白色 16px, 字重 500
- 圆角：16-20px
- 无描边

**颜色系统** | 用途 | 颜色值 | |------|--------| | 主背景 | #FFFFFF | | 统计区背景 | #F8FAFC | | 主文字 | #1A1A1A | | 次要文字 | #6B7280 / #9CA3AF | | 强调色 | #8B5CF6 (紫色认证徽章) | | 状态色 | #22C55E (绿色在线) | | 分隔线 | #E2E8F0 |

**字体层级**

- H1 用户名：22px / 700
- H2 数值：24px / 600
- Body 职位：16px / 400
- Caption 标签：14px / 400

**设计关键词** `clean`, `minimal`, `professional`, `modern`, `card-ui`, `profile-card`, `glass-effect`, `soft-shadow`, `well-spaced`, `hierarchical-typography`

## 2. 产品分享卡片 (Product Share Card)

参考：`src/components/product-card/product-share-card.vue`

### 深蓝色调液态玻璃拟态风格

**配色系统**

- 背景渐变：#0a1628 → #1e3a5f → #0f2942
- 蓝色渐变高光：#3b82f6 → #06b6d4 → #60a5fa
- 卡片玻璃：rgba(255,255,255,0.05) + backdrop-filter: blur(10px)

**布局结构**

- 卡片比例：3:4 (400×533px)
- 头部品牌区：~50px
- 图片区域：240px
- 信息区域：flex: 1 自适应
- 底部联系区：~60px

**核心特性**

- 多层毛玻璃效果 (backdrop-filter: blur(20px))
- 流动渐变背景动画
- 彩虹光晕边框效果
- 悬浮时液态变形动效

## 3. 通用卡片设计原则

### 间距系统

- 卡片内边距：24px
- 区域间距：16-20px
- 元素间距：8-12px
- 紧凑间距：4-6px

### 圆角系统

- 卡片外圆角：24-32px
- 内部容器圆角：16-20px
- 小元素圆角：8-12px
- 圆形元素：50%

### 阴影层级

```scss
// 层级 1 - 轻微浮起
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04);

// 层级 2 - 标准卡片
box-shadow: 0 4px 24px rgba(0, 0, 0, 0.08);

// 层级 3 - 强调卡片
box-shadow: 0 8px 40px rgba(0, 0, 0, 0.12);

// 层级 4 - 悬浮状态
box-shadow: 0 12px 56px rgba(0, 0, 0, 0.16);
```

### 字体比例

```scss
// 标题层级
text-2xl: 24px / 700  // 主要数值/强调
text-xl:  22px / 700  // 用户名/主标题
text-lg:  18px / 600  // 副标题
text-base: 16px / 400 // 正文
text-sm:  14px / 400  // 标签/辅助文字
text-xs:  12px / 400  // 说明文字
```
